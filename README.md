# Habit Tracker · 个人定制化打卡页面

一个基于《逆龄大脑》12 周计划理念打造的**个人定制化打卡工具**。完全在浏览器本地运行，无需注册账号，数据安全存储在你自己电脑上。

## 功能特色

- **双重模式**：设置模式自定义计划内容 + 打卡模式日常使用
- **完全定制**：自定义计划名称、周数、起始日期、分类、任务名、颜色、图标、**每周次数**
- **频率展开**：设置"核心 ×3/周"，打卡表自动展开 3 列，逐次打卡
- **模板系统**：保存/加载/导出/导入计划模板，一键分享给朋友
- **数据持久化**：自动保存到浏览器本地，关闭页面不丢失
- **离线可用**：首次加载后，断网也能正常打卡
- **导出备份**：支持导出/导入完整数据（含配置和打卡记录）
- **响应式设计**：桌面、平板、手机均适配

## 快速开始

### 本地使用

直接用浏览器打开 `cyber.html` 即可使用，无需任何服务器。

### 在线部署（分享给朋友）

#### 方式一：GitHub Pages（推荐）

1. Fork 或克隆本仓库
2. 推送到你的 GitHub 仓库
2. 进入仓库 Settings → Pages → 选择 `main` 分支 → 保存
3. 等待 1-2 分钟，访问 `https://你的用户名.github.io/仓库名/`

> 本项目已配置 GitHub Actions，推送 `main` 分支后会自动部署到 Pages。
>
> 访问地址：`https://你的用户名.github.io/仓库名/`（`cyber.html` 已重命名为 `index.html`）

#### 方式二：任意静态托管

直接将 `cyber.html` 上传到任何静态文件托管服务（如 Vercel、Netlify、Cloudflare Pages 等）。

## 使用指南

### 首次使用

1. 打开页面，默认进入**打卡模式**
2. 点击顶部「设置模式」进入设置
3. 自定义计划名称、周数、起始日期
4. 调整分类和任务（可改名称、颜色、图标）
5. 点击「打卡模式」回到主界面开始打卡

### 模板分享

1. 在设置模式下，点击「保存为模板」保存当前配置
2. 点击「导出模板」下载 `.json` 文件
3. 将文件分享给朋友
4. 朋友点击「导入模板」即可使用

### 数据备份

- 打卡模式下点击「导出数据」可备份所有记录（含配置）
- 点击「导入数据」可恢复备份

## 技术栈

- 纯原生 HTML + CSS + JavaScript
- [FontAwesome 6](https://fontawesome.com/) 图标库（CDN 加载）
- localStorage 数据存储
- Service Worker 离线缓存
